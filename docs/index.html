Sacred Timeline protects your work like git protects code — without needing to know git exists. Just copy the prompt below and paste it into Claude.

New to this? Here's what these words mean

Git A system that saves every version of your work so you can never lose anything or break anything permanently. Like an undo button that goes back months.
GitHub A free website that stores your work safely in the cloud. Think of it as Google Drive, but designed for things built with AI. Create a free account →
Sacred Timeline A human-friendly layer on top of git. Instead of technical commands, you just say: capture, experiment, backup. Claude speaks it fluently. See the project →

Open Claude Code (or Claude.ai)

Any Claude session works. Claude Code gives you the full experience — it can run commands for you.

2

Copy this prompt

This tells Claude exactly what you need — no technical knowledge required.

Paste into Claude
I want to start using Sacred Timeline to protect and track my work. I don't know git — please guide me through setup in plain language. First, check if Sacred Timeline is installed by running: command -v sacred If it's not installed, install it with: npm install -g @suhit/sacred-timeline Then guide me through these steps: 1. Run `sacred status` and tell me what it shows in plain English 2. If I'm not in a project folder yet, help me navigate to one. If the folder isn't a Sacred Timeline yet, run `sacred start` to initialise it. 3. Walk me through my first capture. The correct syntax is: sacred capture "my message here" (just a plain message in quotes — no flags like -m) 4. Explain what backup means and how to connect to cloud. To connect to GitHub the command is: sacred connect Help me find my GitHub URL if I don't know it. Important: only use sacred commands (capture, experiment, backup, latest, connect, status, timeline, narrate). Do not use raw git commands or invent flags that aren't in these examples. Use simple language throughout — pretend I've never heard of git.

Then press Enter — Claude takes it from there.

3

Follow Claude's guidance

Claude will install Sacred Timeline, show you your status, and walk you through your first capture. You're protected from this point forward.
